---
title: Spicy Tofu with Peanuts Sauce
image: "./image.jpg"
published: true
ingredients:
  - name: Smoked tofu cut in dice
    variable: 200
    unit: gr
  - name: Vegetal oil
    variable: 1
    unit: Tbsp
  - name: Shallot
    variable: 1
  - name: Garlic Clove (minced)
    variable: 1
  - name: Soy Sauce
    variable: 1
    unit: Tbsp
  - name: Ketchup
    variable: 1
    unit: Tbsp
  - name: Peanut Butter
    variable: 2
    unit: Tbsp
  - name: Maple Syrup
    variable: 1.5
    unit: tsp
  - name: Vegetable Stock Cube Broth
    variable: 125
    unit: ml
  - name: Sriracha Sauce
    variable: 0.5
    unit: tsp
---

1. In a pan, heat the oil
2. Add the shallots and minced garlic. Brown it 2 minutes
3. Add soy sauce, ketchup, peanut butter, maple syrup, vegetable broth and sriracha sauce. Stir.
4. Add the tofu, let it boil a bit and let it rest until the sauce is creamy.
5. Serve with rice or pasta. Put some cilantro if you like it.
